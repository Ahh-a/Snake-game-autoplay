# 🐍 Snake SDL2 com Ciclo Hamiltoniano

Um clone clássico do jogo Snake feito em **C** usando **SDL2** e **SDL_ttf**, com a particularidade de que a cobra segue um **Ciclo Hamiltoniano** pré-computado.
